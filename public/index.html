Upload a Hitters Stats File

1. Open the Google stats file
2. Select the file tab
3. Select download
4. Select "Comma-Seperated Value (.csv)"
5. The data file should be downloaded on your local computer as a .csv file
6. Upload that file

Select the Stat You Want to Visualize and Print Out

*Once the graph is generated, hover over the top right portion of the graph and
select the camera to download the graph*
